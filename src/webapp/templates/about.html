About the System

Overview

The Ensemble-Based Multi-Disease Prediction System is a machine-learning-powered web application designed to predict multiple diseases using a single user-uploaded CSV file containing patient health attributes.

The system uses ensemble models trained on medical datasets, and provides interpretability using SHAP-based feature explanations. Users receive disease-wise probabilities along with detailed PDF reports.

Diseases Predicted

The system currently supports prediction of the following diseases:

  • Diabetes – based on glucose, BMI, pregnancy count, age, and other metabolic features.
  • Heart Disease – based on chest pain type, blood pressure, cholesterol, ECG data, etc.
  • Kidney Disease – based on hemoglobin level, blood pressure, albumin, etc.
  • Parkinson’s Disease – based on vocal measurements such as jitter, shimmer, and frequency variations.

Key Features of the System

  • Simple and user-friendly CSV upload interface
  • Highly accurate ensemble ML prediction models
  • Detailed SHAP-based textual explanations
  • Downloadable PDF reports for each prediction
  • Admin dashboard for model retraining and user monitoring

Who Can Use This System?

The platform is designed for:

  • General users who want to self-check medical risk levels
  • Doctors and medical students for analysis and study
  • Researchers and ML enthusiasts exploring medical predictions
